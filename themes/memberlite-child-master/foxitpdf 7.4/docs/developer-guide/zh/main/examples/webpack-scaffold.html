# 脚手架工程

此demo是UIExtension的脚手架工程。源码中包含了一个完整的开源addon。

examples/UIExtension/scaffoldDemo/

# 如何运行

# 设置库

在demo目录下执行npm run setup 命令。

该命令执行了以下步骤

  • 将 UIExtension 包作为本地依赖.
  • 安装其他npm依赖.

# 启动

执行命令 npm start

# 源码结构

src文件夹包含如下内容：

│  addons.js                        --- 展示如何使用foxit提供的插件.
│  index.js                         --- javascript入口文件.
│
└─addonExample                      --- addon的样例. 
    │  addon.info.json              --- addon入口文件, 该文件指定了所有addon需要的其他文件.
    │  index.css                    --- 样式表
    │  index.js                     --- addon的脚本入口文件。不要使用其他文件名。
    │
    ├─locales                       --- i18n 多语言文件
    │      en-US.json
    │      zh-CN.json
    │
    ├─stateHandlers                 --- 继承自IStateHandler的State Handler 类 
    │      addTextField.js
    │
    └─templates
            custom-dialog.art       --- 自定义对话框的art 模板.
            tab-template.art        --- 自定义toolbar的art模板.
