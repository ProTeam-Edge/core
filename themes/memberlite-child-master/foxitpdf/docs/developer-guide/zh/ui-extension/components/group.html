# Group Component

Group 组件通常用在 toolbar 上，它的作用是给组件用一条竖线分隔，每一组表示一个类别，同时可以设置在小的屏幕下进行收缩以尽可能多的显示组件的功能。

# 代码演示

# 指定收缩时机

# 指定收缩后预留的组件数量

                <group-list shrink-size="762">

                    <group name="home-tab-group-hand" retain-count="2">
                        <hand-button></hand-button>
                        <selection-button>Selection</selection-button>

# API

# 组件模板

<group-list shrink-size="762">
    <group retain-count="2" shrink-title="options"></group>
    <group retain-count="1" shrink-title="options"></group>
</group-list>

group-list 模板属性说明：

属性 说明 类型 默认值 版本
shrink-size 指定一个像素宽度，<webpdf> 组件宽度小于这个宽度时触发收缩 number 1024 7.0.0

group 模板属性说明：

属性 说明 类型 默认值 版本
retain-count 指定收缩后，保留的组件数量 number 1 7.0.0
shrink-title 指定收缩后，下拉列表顶部的标题内容 string '' 7.0.0
shrink-size 覆盖group-list的shrink-size值, 指定一个像素宽度，<webpdf> 组件宽度小于这个宽度时触发收缩 number 父组件的shrink-size值 7.0.0

# 方法

group 组件方法

方法定义 说明 版本
setRetainCount(count: number): void 设置收缩保留组件数 7.0.0
setShrinkTitle(title: string): void 设置收缩列表标题 7.0.0

# Events

<group> 组件事件

事件名称 说明 示例 版本
shrink 收缩或展开时触发 group.on('shink', (isShrinked) => void) 7.4.0

<group-list> 组件事件

事件名称 说明 示例 版本
shrink 收缩或展开时触发 groupList.on('shink', (groupComponent, isShrinked) => void) 7.4.0
